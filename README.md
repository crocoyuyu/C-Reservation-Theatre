# C-Reservation-Theatre  
- Projet en cours de conception   
Création d'un système de gestion des places de théâtre de la COMEDIE FRANCAISE. Les représentations sont connues pour toute la saison, et le logiciel doit gérer une liste d'abonnés.

Dans le cadre du cours de Langage C à l'Université Paris Cité

Enseignant : Jean-Michel Léry

Réalisé par : Yuan Ding & Amira Kairouani
